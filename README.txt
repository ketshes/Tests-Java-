
Как развернуть тестовую среду на новой машине:

Установить:

1.ОС Ubuntu версия 16.04

2.Eclipse IDE версия neon 2.0

3.JUnit 4.10

4. Mozilla Firefox 45.05.
Важно! запретить firefox автоматически обновляться, т.к. это может привести к сбоям в работе!

5. geckodriver

4.Selenium IDE 3.0.1

!!!!
5.В приложенных тестах:

в зависимости от конфигурации изменить absoluteFilePath в классе TestRunner.
